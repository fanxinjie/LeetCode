链表的题目种类较为单一，链表的问题常见的技巧是：

- 画图，多用几个变量可能会有助于指针移动，使得问题简单；
- 设置虚拟头节点，避免分类讨论；
- 编写简单的方法（打印链表、通过数组构造链表），帮助测试；
- 快慢指针: 这一类问题特别典型，如果不学习几乎很难想到，但是也没有什么创新的地方，知道怎么做就行了。

2. 穿针引线：206、92、328
3. 设置虚拟头节点：203、328、83、82、21、2
4. 快慢指针：19、61、141、143、234
